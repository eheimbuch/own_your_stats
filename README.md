# own_your_stats 🚀

Skill-Datenbank für Unternehmen — **offline-first** mit maximaler Performance.

## Features

- **Skills verwalten** — Fachliche und nicht-fachliche Skills mit Kategorien
- **Selbsteinschätzung** — Level 1-5, Interesse, Berufserfahrung
- **Fremdbewertung** — Anonyme Bewertungen von Kollegen
- **Synergien finden** — Wer kann was? Match mit Projekten
- **Offline-First** — Vollständig nutzbar ohne Internet (IndexedDB + PWA)
- **Installierbar** — Wie eine native App auf Desktop & Mobil

## Tech Stack

| Layer | Technologie |
|---|---|
| Framework | React 18 + TypeScript |
| Build | Vite 5 |
| Styling | Tailwind CSS 3 |
| DB | Dexie.js (IndexedDB) |
| State | Zustand |
| PWA | vite-plugin-pwa (Service Worker) |
| Icons | Lucide React |

## Entwicklung

```bash
npm install
npm run dev     # Entwicklungsserver
npm run build   # Produktionsbuild
npm run preview # Build preview
```
